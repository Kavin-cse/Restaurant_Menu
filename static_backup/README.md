# Dhabi Restaurant Landing Page

A luxurious, 3D-rendered landing page for **Dhabi Restaurant**.

## Features
- **Design**: Premium "Zero Gravity" aesthetic with dark themes and golden accents.
- **Tech Stack**: HTML5, CSS3, JavaScript (Vanilla).
- **Libraries**: [GSAP](https://greensock.com/gsap/) for smooth animations and scroll effects.
- **Assets**: AI-generated high-fidelity 3D food renders.

## How to Run
1. **Simple Method**: Double-click `index.html` to open it in your browser.
2. **Best Method**: Use a local development server (e.g., Live Server in VS Code) for correct asset loading and smoother performance.
   - If you have Python installed: `python -m http.server`
   - If you have Node/NPM: `npx serve .`

## Sections
1. **Hero**: 3D Thali platter with floating ingredients and mouse parallax effect.
2. **Menu**: 3D floating cards for dishes with hover tilt effects.
3. **About**: Sambar bowl visual.
4. **Chef**: Animated chef character with floating vegetables.
5. **Testimonials**: Customer reviews.
